# Проект автоматизации тестирования Кинопоиска

## Описание проекта

Проект содержит автоматизированные тесты для сайта Кинопоиск и API сервиса Кинопоиска.

Реализованы:

- UI тесты с использованием Selenium
- API тесты
- отчеты тестирования с использованием Allure

Тесты проверяют:

- получение информации о фильме
- редирект на страницу авторизации для неавторизованного пользователя
- работу API метода получения информации о фильме

---

## Используемые технологии

- Python
- pytest
- Selenium WebDriver
- requests
- Allure Report

## Запуск тестов

- запуск всех тестов: python -m pytest
- запуск только UI тестов: python -m pytest test/test_ui.py
- запуск только API тестов: python -m pytest test/test_api.py

При запуске UI тестов, предусмотрено время для закрытия рекламных окон и прохождения капчи
